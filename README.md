# smallbiz_front

A sample Vue-based website demo. It is mobile-friendly and responsive. Uses SASS, the CSS preprocessor.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
